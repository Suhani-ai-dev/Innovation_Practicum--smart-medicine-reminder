# 💊 Smart Medicine Reminder System
 
## 🧠 Overview
This is a software prototype of a Smart Medicine Dispensing & Monitoring System.  
It helps users remember to take medicines on time and track their medication.
 
---
 
## 🚀 Features
- ⏰ Medicine scheduling
- 🔔 Reminder alerts
- 🔊 Voice notifications
- ✅ Take / ❌ Miss tracking
- 📊 Dashboard (Total, Taken, Missed)
- 💾 Data saved using LocalStorage
 
---
 
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
 
---
 
## ⚙️ How It Works
1. User adds medicine details (name, time, dose)
2. System triggers reminder at correct time
3. Voice alert notifies the user
4. User marks medicine as Taken or Missed
5. Dashboard updates automatically
 
---
 
## 🔗 Future Scope
- Integration with Arduino/ESP32
- Automatic pill dispensing system
- Mobile application
- AI health assistant
 
---
 
## 🎯 Purpose
This project demonstrates how software can be used to build a smart healthcare solution and can be extended with hardware components.
